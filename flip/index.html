Construyendo FLIP

Cómo lo construimos

I/O FLIP es un ejemplo de integración perfecta entre las nuevas herramientas de inteligencia artificial de Google y las herramientas que ya conoce y ama. Las imágenes de personajes se generan con DreamBooth en Muse y las descripciones se generan utilizando la API PaLM. La interfaz de usuario del juego y las animaciones se crean en Flutter y el backend está escrito en Dart. Utiliza un conjunto de herramientas de Firebase y Google Cloud para alojar y compartir.

Preguntas frecuentes

Preguntas frecuentes

¿Qué es esto?

I/O FLIP es un juego de cartas diseñado por IA impulsado por Google, creado para Google I/O 2023.

¿Cómo se crearon las imágenes y descripciones?

Las imágenes se generaron utilizando Muse, un modelo que transforma de texto a imagen desarrollado por Google Research, y se entrenaron utilizando el arte que creamos. El texto se generó utilizando la API PaLM basada en entradas de texto creadas por Google. Para garantizar una experiencia positiva para todos los usuarios, las imágenes y el texto se generaron fuera de línea para que revisores humanos pudieran examinarlos y garantizar que cumplieran con los principios de IA de Google.

¿Cómo se utilizó la IA?

Se utilizó IA para crear las imágenes y descripciones de cada tarjeta.

¿Hay premios para los ganadores?

No hay premios como parte del juego.

¿Qué datos se almacenan y están disponibles públicamente cuando se comparten?

Si elige compartir, sus iniciales, tarjetas y puntaje estarán disponibles en una URL única durante 30 días y luego se eliminarán automáticamente.
