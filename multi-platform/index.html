Empezar

Hermosas aplicaciones para cada pantalla

Flutter te permite crear aplicaciones para dispositivos móviles, web, de escritorio y integrados, todo desde una única base de código.

Base de código única

Mantenga una base de código e impleméntela en múltiples plataformas, acelerando y simplificando los flujos de trabajo.

Rendimiento por diseño

Flutter te brinda el poder de los gráficos acelerados por hardware para aplicaciones de alto rendimiento en cualquier plataforma.

Personaliza cada píxel

El motor de renderizado Flutter te permite controlar cada píxel y su biblioteca de widgets se adapta automáticamente a cualquier pantalla.

Móvil

Aplicaciones para iOS y Android

Cree funciones una vez e impleméntelas tanto en iOS como en Android. Los diseños de Cupertino y Material están integrados en el marco Flutter, por lo que tus aplicaciones se sienten como en casa en ambas plataformas.

Flutter on mobile

Web

Aplicaciones Web

Llegue a usuarios de todas partes implementando aplicaciones Flutter en la web. Cree prototipos rápidos e implemente su aplicación móvil en la web desde la misma base de código.

Flutter en la web

Escritorio

Aplicaciones de Windows, macOS y Linux

Transforme sus aplicaciones en experiencias de escritorio con una única base de código y herramientas familiares. Apunte a Windows, macOS y Linux sin reescribir.

Flutter en el escritorio

Integrado

Aplicaciones Flutter, en cualquier lugar

Cree soluciones personalizadas con el poder y la flexibilidad de Flutter. Implemente en cualquier lugar, incluidos dispositivos inteligentes, automóviles y más.

Flutter en dispositivos integrados
