Crear widgets de pantalla de inicio y de bloqueo para tu aplicación Flutter

En este taller, recorreremos los pasos necesarios para agregar widgets de pantalla de inicio y widgets de pantalla de bloqueo a su aplicación Flutter en iOS y Android.


Antes de llegar

Este taller recorrerá el desarrollo nativo de Android e iOS. Además de tener instalado el SDK de Flutter y un entorno de desarrollo, necesitarás cierto software para seguir cada parte. Para seguir la parte de Android, solo necesitarás Android Studio . Para seguir la parte de iOS, necesitará una computadora MacOS con Xcode instalado y una cuenta de desarrollador de Apple . Puede encontrar más información en el archivo README de este repositorio de GitHub .

Práctica con autenticación multifactor y consultas de recuento de Firestore

Obtén las últimas funciones de Firebase para tu aplicación Flutter. Comenzaremos con una aplicación básica y luego agregaremos autenticación telefónica multifactor para garantizar un inicio de sesión más seguro. Luego usaremos el nuevo operador COUNT() en Firestore para incluir recuentos en la aplicación en solo una fracción del costo y complejidad.


Antes de llegar

Para poder seguir este taller, necesitará configurar un entorno de desarrollo de Flutter y Firebase. Puedes seguir las instrucciones de configuración en el laboratorio Conozca Firebase para Flutter . Es importante que complete los pasos del uno al cuatro del laboratorio para configurar su entorno. Si no tiene experiencia en Firebase, le recomendamos que complete todo el laboratorio de código antes del taller.

Este taller incluye pasos del proceso de diseño, que ayudarán a los desarrolladores a aprender cómo convertir el diseño en código. Aparte de Figma, este taller solo utilizará el entorno de desarrollo estándar Flutter.
